# pockemonAPI


Интерфейс поиска
Вот как устроен интерфейс:

Панель сортировки: Вы можете отсортировать список покемонов по ID, имени, HP, атаке или защите. Также доступен выбор порядка — от меньшего к большему или наоборот.

Блок фильтров: Позволяет искать покемонов по имени (текстовое поле) и задавать диапазоны для HP, атаки и защиты с помощью ползунков. Значения диапазонов отображаются в реальном времени.

Карточки покемонов: Каждая карточка показывает имя, ID, тип (с цветовой меткой) и три основных параметра с визуальной шкалой. При наведении карточка приподнимается и подсвечивается.

Кнопка «Применить»: Запускает фильтрацию и сортировку на основе выбранных параметров, обновляя список результатов.
